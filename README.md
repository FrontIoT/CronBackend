# Cron Backend

## Description

    This library creats an easy way to schedule BusinessProcess in your application through a crontab-like configuration file.
    
## Usage

    Use CronBackend\CronBackendBase.pkg
    
    Class cCronBackend is a cCronBackendBase
    End_Class
    
    Object oCronBackend is a cCronBackend
    End_Object
    
    Send StartCronProcess of oCronBackend
